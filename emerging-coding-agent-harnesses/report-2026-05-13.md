## 1. gptme (Erik Bjäreholt)

- **Source:** https://github.com/gptme/gptme (HTTP 200 ✓)
- **Stars:** 4.3k
- **Auth:** API key via environment variable (`OPENROUTER_API_KEY`, `ANTHROPIC_API_KEY`, etc.)
- **Non-interactive:** ✅ `gptme -n "task"` (non-interactive mode flag)
- **Autonomous:** ✅ `gptme-agent-template` provides systemd/launchd run loops; Bob agent runs 1700+ autonomous sessions
- **VM compatibility:** ✅ pipx install, Python 3.10+
- **Evidence:** Prior autonomy success — controlled bugfix in supervisor-agent (26s, $0.0022, correct fix, no phantom changes)
- **Verdict:** ADOPT. Already installed and validated. Best candidate for deepening (multi-file bugfix, autonomous run loop).

- **Source:** https://github.com/ai-christianson/RA.Aid (HTTP 200 ✓)
- **Docs:** https://docs.ra-aid.ai/ (HTTP 200 ✓)
- **Stars:** 2.2k
- **Auth:** API key via environment variable (`ANTHROPIC_API_KEY`, `OPENAI_API_KEY`, `OPENROUTER_API_KEY`)

- **Verdict:** ADOPT. Best next smoke-test target. `--cowboy-mode` is explicitly designed for non-interactive use. Install and run on a controlled bugfix in high-signal-news or skill-validation lab.
